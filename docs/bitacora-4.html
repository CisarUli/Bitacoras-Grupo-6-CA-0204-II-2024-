4.2.3 M√©todo #1: Regresi√≥n Lineal Simple**

: La regresi√≥n lineal simple (Desconocido 2024) es un modelo estad√≠stico usado para relacionar una variable independiente X con una variable dependiente Y. Es decir, en una regresi√≥n lineal simple solo hay dos variables (la variable explicativa X y la variable respuesta Y) y se intenta aproximar la relaci√≥n que hay entre ambas variables.

Por lo tanto, la regresi√≥n lineal simple sirve para encontrar una ecuaci√≥n que relacione dos variables de una manera lineal. L√≥gicamente, la relaci√≥n entre las dos variables debe ser lineal, sino se debe utilizar otro tipo de modelo de regresi√≥n.

La ecuaci√≥n de un modelo de regresi√≥n lineal simple est√° formada por dos coeficientes: la constante de la ecuaci√≥n (b0) y el coeficiente de la correlaci√≥n entre las dos variables (b1). Para poder hacer una regresi√≥n lineal simple, se deben cumplir las siguientes hip√≥tesis:

  • Independencia: los residuos observados deben ser independientes entre s√≠. Una manera com√∫n de garantizar la independencia del modelo es a√±adiendo aleatoriedad en el proceso de muestreo.

  • Homocedasticidad: debe haber homogeneidad en las varianzas de los residuos, es decir, la variabilidad de los residuos debe ser constante.

  • Normalidad: los residuos deben estar distribuidos normalmente, o dicho de otra forma, deben seguir una distribuci√≥n normal de media 0. Linealidad: la relaci√≥n entre la variable independiente y la variable dependiente debe ser lineal.
